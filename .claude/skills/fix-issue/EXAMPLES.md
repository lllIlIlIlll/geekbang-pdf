# Fix Issue 使用示例

## 示例 1：PDF 生成空白

```
用户：PDF 生成出来是空白的
助手：使用 fix-issue skill
→ 检查 Cookie 是否有效
→ 验证页面是否正确加载
→ 确认内容容器是否展开
```

## 示例 2：批量下载失败

```
用户：批量下载中断了
助手：使用 fix-issue skill
→ 检查 urls_batch.txt 格式
→ 验证网络连接
→ 查看中断位置的日志
```

## 示例 3：登录后无法保存

```
用户：登录成功了但是 Cookie 没保存
助手：使用 fix-issue skill
→ 检查写入权限
→ 验证配置文件路径
→ 确认 Cookie 格式正确
```

## 示例 4：页面结构变化

```
用户：某个课程页面无法下载
助手：使用 fix-issue skill
→ 对比正常页面结构
→ 更新 selectors.json
→ 测试修复
```
